# **HƯỚNG DẪN CẤU HÌNH**

**Khởi tạo các Packages Composer**

```console
composer i
```
**Khởi tạo các Packages NPM**
```console
npm i
```

**ĐỔI FILE .env.example thành .env**

**TẠO KEY**
```console
php artisan key:generate
```

**MIGRARE database**
```console
php artisan migrate:fresh --seed
```
